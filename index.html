Kinetik Kum Sanatı Masası

Sisyphus tipi, Ø60 cm polar (theta-rho) tasarım — fikirden üretilebilir ürüne: BOM, 3D model, PCB (Gerber + STEP) ve fotogerçekçi render.

Ø60 cm · polar ESP32 · Dune Weaver PCB: 0 ratsnest · 0 short Blender · Cycles

Reklam Filmi

Kamera yörüngede dönerken bilye "K. Atatürk" imzasını çiziyor, LED yanıyor — Blender / Cycles, post (bloom).

İnteraktif 3D Simülasyon

Gerçek makine gibi: tıkla → LED yanar, bilye .thr yörüngesinde gezerek deseni çizer. LED modları (mavi/sıcak/gökkuşağı), desen seçimi (Atatürk imzası/spiral/klasik) ve patlatma ile tüm alt bileşenleri gör.

🧊Tam ekran aç
Three.js · table.glb (16 parça) · canlı çizim + LED

Fotogerçekçi Render (Blender / Cycles)

Ayaklı sehpa · gerçek .thr Sandify deseni · oturma odası HDRI · PBR malzemeler (render/scene.py).

Desen (üstten) — Atatürk imzası kuma çizilmiş, ucunda çelik bilye
Ürün çekimi — mat ceviz sehpa, açık kum yüzey
Oturma odası — HDRI ortam, cam üst görünür
Ambiyans — koyu sahne, LED halka parıltısı

Kontrol Kartı (PCB)

KiCad 9 + Freerouting ile tam routing — DRC: 0 ratsnest 0 kısa devre 0 bakır clearance

Üst bakır — sinyal routing (139 iz + 3 via)
Alt bakır — GND ground plane (clearance'lı)
Montaj çizimi — ESP32 + 2× TMC2209 + buck + klemensler
